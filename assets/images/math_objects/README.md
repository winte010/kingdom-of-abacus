# Math Objects Assets

This directory contains visual assets for interactive math objects.

## Required Images

The following PNG images should be placed in this directory:

### shell.png
- **Recommended size**: 200x200 pixels
- **Description**: A colorful seashell icon for counting and grouping activities
- **Suggested colors**: Blues, purples, sandy colors
- **Style**: Friendly, cartoon-style illustration

### treasure.png
- **Recommended size**: 200x200 pixels
- **Description**: A treasure item (gem, coin, or jewel) for collection activities
- **Suggested colors**: Gold, gems colors (red, blue, green)
- **Style**: Sparkling, appealing to children

### rock.png
- **Recommended size**: 200x200 pixels
- **Description**: A smooth rock or stone for stacking activities
- **Suggested colors**: Gray, brown, earth tones
- **Style**: Simple, natural-looking stone

### chest.png
- **Recommended size**: 200x200 pixels
- **Description**: A treasure chest for collecting items
- **Suggested colors**: Brown wood, gold accents
- **Style**: Classic treasure chest design

## Fallback Behavior

If these images are not present, the widgets will automatically fall back to using Material Icons:
- Shell → Water drop icon
- Treasure → Diamond icon
- Rock → Circle icon
- Chest → Inventory icon

## Creating Placeholder Images

To create simple placeholder images, you can:

1. Use any image editor (GIMP, Photoshop, Figma, etc.)
2. Create 200x200px images with transparent backgrounds
3. Draw simple representations of each object
4. Export as PNG with transparency
